# PHP + Tailwind Starter Template

A clean, high-performance boilerplate for building PHP applications with Tailwind CSS.

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   composer install
   npm install
   ```

2. **Development**:
   ```bash
   npm run dev
   ```

3. **Production Build**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `pages/`: Add your `.php` page templates here. Routing is automatic (e.g., `pages/about.php` is accessible at `/about`).
- `include/`: Shared components like `header.php` and `footer.php`.
- `assets/`: Static files (images, JS, fonts).
- `src/`: Tailwind source file (`input.css`).
- `dist/`: Compiled production assets.

## 🛠️ Tech Stack

- **PHP 8.x**
- **Tailwind CSS v3**
- **PostCSS**
- **Composer & npm**
